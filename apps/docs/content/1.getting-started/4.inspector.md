该模块内置了与 [MCP Inspector](https://github.com/modelcontextprotocol/inspector) 的集成，这是一个可视化调试工具，允许您直接在 Nuxt DevTools 中以交互方式测试和调试您的 MCP 服务器。

:u-color-mode-image{light="/mcp-devtools-light.png" dark="/mcp-devtools-dark.png" alt="MCP Inspector 预览" class="w-full rounded-lg my-6 grayscale"}

## 使用方法

1. **启用 DevTools** - 确保在 `nuxt.config.ts` 中启用了 DevTools：
   ```typescript [nuxt.config.ts]
   export default defineNuxtConfig({
     modules: ['@nuxtjs/mcp-toolkit'],
     devtools: { enabled: true },
   })
   ```

2. **启动** - 打开 Nuxt DevTools 并导航至 **Server** 部分下的 **MCP Inspector** 选项卡，然后点击 **Launch Inspector**。

3. **测试** - 使用检查器浏览工具、资源和提示词，使用自定义参数进行测试，并查看请求/响应历史记录。

检查器会自动使用正确的配置连接到您的 MCP 服务器端点——无需额外设置。

## 为什么使用它？

检查器提供了一个可视化界面，用于：
- 浏览所有可用的工具、资源和提示词
- 使用自定义参数测试工具并查看响应
- 查看请求/响应历史记录以进行调试
- 调试连接问题和错误

这是在开发过程中测试和验证 MCP 服务器最简单的方法。

## 配置

默认情况下，检查器使用端口 `6274`（UI）和 `6277`（代理）。如需自定义它们：

```bash
CLIENT_PORT=8080 SERVER_PORT=9000 npm run dev
```

有关高级配置选项，请参阅 [MCP Inspector 文档](https://github.com/modelcontextprotocol/inspector)。
